# Training module "Approche Écosystémique des Ressources Marines Exploitées" (AERME)

Presentation for the session 2022 is available through the following link: https://ob7-ird.github.io/training_aerme/.
You can also generate it by running the quarto file (you need for that to install the Quarto software https://quarto.org/docs/get-started/) or by running the following code:

``` r
install.packages("pagedown")
pagedown::chrome_print(input = "https://ob7-ird.github.io/training_aerme/#/section.html")
```

Materials related to the role play are in the directory role_play_rcg. The zip archive stored in the directory is protected by a password to avoid any spoil regarding the simulation.
